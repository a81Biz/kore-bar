Proyecto de Tesis

Kore Bar OS

Sistema Integral de Gestión para Restaurantes. Arquitectura orientada a Micro-Frontends, sincronización de datos en tiempo real y despliegue modular.

Micrositios del Sistema

admin_panel_settings

Administración Web

Centro de control gerencial. Gestión de piso (mesas y zonas), directorio de empleados sincronizado con RRHH, configuración de menú y reportes de ventas.

Lanzar Módulo
restaurant

Pizarra de Cocina

Interfaz táctil para recepción de comandas en tiempo real. Algoritmo de priorización por hora/mesa y gestión de aprovisionamiento de inventario.

Lanzar Módulo
room_service

Terminal Meseros

Aplicación optimizada para tabletas. Levantamiento de pedidos, notificaciones PUSH de platillos listos y solicitud de cierre de cuentas.

Lanzar Módulo
point_of_sale

Caja y Facturación

Punto de venta (POS) para recepción de pagos. Incluye integración con el catálogo del SAT para emisión y timbrado de facturas electrónicas (CFDI).

Lanzar Módulo
qr_code_2

Menú QR Comensales

Interfaz de solo lectura, adaptativa a dispositivos móviles (Mobile-First). Consume el catálogo centralizado de platillos y precios en tiempo real.

Lanzar Módulo
receipt_long

Portal de Facturación

Portal público de auto-servicio. Permite a los comensales solicitar su comprobante fiscal (CFDI 4.0) ingresando el folio único de su ticket de consumo.
